Introducción

La última semana que contendrá esta bitácora. Ha sido un arduo camino lleno de aprendizajes inesperados, desde entender qué es GitHub y cómo usarlo, hasta aprender HTML para crear páginas web. He aprendido a utilizar Visual Studio Code para organizar mi trabajo y mantener una bitácora estructurada, y también he creado un sitio web para un proyecto innovador, aplicando lo aprendido y explorando nuevas posibilidades. Ver el sitio web tomar forma ha sido muy satisfactorio, y esta experiencia ha sido clave en mi desarrollo personal y profesional.

Clase Presencial

Como último gran aprendizaje de este curso, aprendimos sobre la realidad aumentada y la realidad virtual.
Realidad Virtual: Es una tecnología que te transporta a un entorno completamente digital, creado por computadora. Usando dispositivos como cascos o gafas especiales, te sumerges en un mundo virtual que te aísla del entorno físico. En este mundo digital, puedes ver, oír y, a veces, incluso tocar elementos que parecen reales, brindándote una experiencia totalmente envolvente.
Realidad Aumentada: Es una tecnología que añade elementos digitales al mundo real. En lugar de reemplazar tu entorno, coloca imágenes, videos o información digital sobre lo que ves a tu alrededor. Utilizas dispositivos como teléfonos inteligentes, tabletas o gafas especiales para ver estas capas digitales que se mezclan con la realidad, creando una experiencia que combina lo físico y lo digital de manera interactiva y complementaria.

Clase Virtual

El profesor quiso que para esta última clase virtual usáramos la plataforma "Miro", ideal para brainstorming, mapas mentales y conceptuales. Nosotros la utilizamos para el brainstorming de nuestros entregables del proyecto final. Durante la sesión, anotábamos nuestras ideas y propuestas en post its, facilitando la discusión y el análisis, mientras el profesor nos daba feedback en tiempo real, guiándonos para estructurar mejor nuestras ideas y cumplir con los requisitos del proyecto. Usar Miro hizo la clase más interactiva y dinámica, enseñándonos una herramienta útil para futuros trabajos colaborativos.
